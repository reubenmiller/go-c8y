# V2

## Goals

* [x] Allow free-form bodies for all POST/PUT commands
* [x] Allow custom response marshalling (with official models that can be used, or the user's own models)
* [x] Allow users to customize single requests (e.g. add their own custom headers/settings etc.)
* [x] Pagination (with IncludeAll...or an iterator etc.)
* [ ] Support authentication handlers
    * [ ] Basic
    * [ ] Internal Token
    * [ ] SSO
* [ ] Auto token renewal
* [ ] Resolve By Name
* [ ] Allow Upsert
* [ ] GetOrCreate commands
