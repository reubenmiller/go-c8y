# V2

## Goals

* Allow free-form bodies for all POST/PUT commands
* Allow custom response marshalling (with official models that can be used, or the user's own models)
* Allow users to customize single requests (e.g. add their own custom headers/settings etc.)
* Pagination (with IncludeAll...or an iterator etc.)
* Auto token renewal
* Resolve By Name
