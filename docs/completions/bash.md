
# Bash completions

## Adding completions to your console's profile

### Bash

Execute the following command to add the completions to your profile:

```sh
echo 'source <(c8y completion bash)' >> ~/.bashrc
```

### zsh

Execute the following command to add the completions to your profile:

```sh
echo 'source <(c8y completion zsh)' >> ~/.zshrc
```
